## Icon systém

BU1SPORT používá **Tabler Icons** jako primární icon set. Prohlížeč a reference: <a href="https://icon-sets.iconify.design/tabler/" target="_blank" rel="noreferrer">icon-sets.iconify.design/tabler</a>

Důvody výběru:
- Konzistentní stroke-based geometrie (stroke-width 2px) a vizuální váha
- Rozsáhlá knihovna — 5 400+ ikon pokrývající e-commerce, sport i obecné UI
- Plná integrace přes Iconify — dostupné ve všech frameworcích
- Open source (MIT licence)

### Vlastní ikony

Pokud Tabler nepokrývá specifický kontext (např. brankářská rukavice, speciální produktový symbol), je možné vytvořit vlastní ikonu. Musí ale **vizuálně vycházet z Tabler setu**:

- Stejný stroke-width: **2px**
- Stejný viewport: **24×24**
- Stejné zaoblení rohů a zakončení linií (round cap, round join)
- Bez fillů — stroke only (výjimka: filled varianta pro active stav)

:::caution[Schválení]
Každou custom ikonu musí schválit brand tým před nasazením do produkce.
:::

## Ikona + text

Ikony se téměř vždy zobrazují spolu s textem — **ikona bez textu vyžaduje `aria-label`**.

<table class="bu1-spec-table">
  <thead><tr><th>Situace</th><th>Pravidlo</th></tr></thead>
  <tbody>
    <tr><td>Ikona + viditelný text</td><td>Ikona je <code>aria-hidden="true"</code></td></tr>
    <tr><td>Ikona bez textu (icon button)</td><td>Musí mít <code>aria-label="Popis akce"</code></td></tr>
    <tr><td>Ikona jako dekorace</td><td><code>aria-hidden="true"</code>, žádný alt</td></tr>
    <tr><td>Ikona komunikuje stav</td><td>Textová alternativa musí existovat (tooltip, sr-only text)</td></tr>
  </tbody>
</table>

## Do / Don't

<div class="bu1-template-grid">
  <section class="bu1-template-section">
    <span class="bu1-template-label">Dělej</span>
    <ul class="bu1-template-list">
      <li>Konzistentní styl (Regular nebo Bold) v celé komponentě</li>
      <li>Velikost ikony odpovídá vizuální váze textu vedle ní</li>
      <li>Ikona + text = ikona je aria-hidden</li>
      <li>Standalone ikona = povinný aria-label</li>
      <li>Fill varianta pro aktivní/selected stav</li>
    </ul>
  </section>
  <section class="bu1-template-section">
    <span class="bu1-template-label">Nedělej</span>
    <ul class="bu1-template-list">
      <li>Nemíchej Tabler s jinými icon sety (FontAwesome, Material, Phosphor)</li>
      <li>Nepoužívej ikony pod 12px — použij text nebo vynech</li>
      <li>Nepoužívej ikonu místo textu pokud je text jasnější</li>
      <li>Nepoužívej Hot Rose na ikony bez akčního kontextu</li>
      <li>Nepoužívej stroke-width jiný než 2px — narušuje vizuální konzistenci</li>
    </ul>
  </section>
</div>

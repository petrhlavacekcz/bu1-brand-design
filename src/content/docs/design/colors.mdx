## Core palette

Tři barvy tvoří základ celého vizuálního systému. Každá má pevně danou roli — nezaměňuj ji.

:::

## Tints & Shades

Hot Rose a Onyx lze derivovat mísením s bílou (tint) nebo černou (shade). Nikdy nepřidávej jinou barvu — zachovej čistotu palety.

    </ul>

:::caution[Pravidlo derivace]
Tints a shades slouží pro UI stavy (hover, active, disabled) a jemné pozadí. Nikdy je nepoužívej jako náhražku core barev v hlavních elementech.
:::

## Žádné gradienty

Gradienty nejsou součástí BU1SPORT design systému. Nepoužívej je nikde — ani v commerce, ani v kampani, ani jako overlay.

<div class="bu1-template-grid">
  <section class="bu1-template-section">
    <span class="bu1-template-label">Místo gradientu použij</span>
    <ul class="bu1-template-list">

      <li>Čistý přechod kontextu přes layout (light sekce → dark sekce)</li>
      <li>Rgba průhlednost pro overlay přes fotku (max 60 % opacity)</li>

  <section class="bu1-template-section">

      <li>Přechod z jedné brand barvy do druhé</li>
      <li>Soft overlay přes produktové fotografie</li>
    </ul>
  </section>
</div>

## Exporty

Canonical zdroj je `src/content/data/brand-tokens.json`.
Odvozené exporty generuj přes `npm run generate:brand`:

- `generated/brand/tokens.css` — CSS custom properties
- `generated/brand/theme.css` — component classes
- `generated/brand/tailwind-preset.ts` — Tailwind konfigurace
- `generated/brand/shopify-tokens.liquid` — Shopify variables
- `generated/brand/figma-variables.json` — Figma tokens

## Metriky

| Vlastnost | Hodnota |
|-----------|---------|
| Min. šířka karty | `200px` |
| Oblast obrázku | `160px` výška |
| Padding těla | `16px` |
| Border | `1px solid Neutral 300` |
| Border radius | `2px` |
| Title font | Oswald, uppercase |
| Title velikost | `0.9rem – 1.25rem` |
| Cena font | Nunito Sans 800 |
| Cena velikost | `1.1rem – 1.375rem` |

## Pravidla

<ul class="bu1-template-list">
  <li>Cena je vidět okamžitě — bez hoveru, bez kliknutí</li>
  <li>Název a cena jsou první dvě věci, které oko zachytí</li>
  <li>Max 2 badge na produkt — badge je signál, ne dekorace</li>
  <li>Žádný dekorativní overlay nebo gradient přes obrázek produktu</li>
  <li>Varianty zobrazuj jako chips, ne jako dropdown pokud je místo</li>
</ul>

## Nedělej

<ul class="bu1-template-list">
  <li>Fashion editorial styl — karta slouží prodeji, ne inspiraci</li>
  <li>Slabá nebo malá cena (nesmí být menší než popis)</li>
  <li>Dekorativní overlay přes produktovou fotografii</li>
  <li>Více než 2 badge — působí chaoticky</li>
  <li>Cena skrytá za hover nebo accordion</li>
</ul>

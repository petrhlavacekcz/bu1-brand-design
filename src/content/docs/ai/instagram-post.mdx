<div class="bu1-section-intro">
  <span class="bu1-kicker">Social</span>
  <p>Instagram může být živější, ale pořád musí být jasný, podpůrný a obchodně použitelný.</p>
</div>

## Povinné vstupy

<ul class="bu1-template-list">
  <li>co přesně se komunikuje</li>
  <li>komu je post určený</li>
  <li>na jakém účtu post vychází</li>
  <li>v jakém jazyce post vychází</li>
  <li>hlavní benefit</li>
  <li>cílová akce</li>
  <li>jestli jde o novinku, edukaci, reference nebo termín</li>
</ul>

## Struktura captionu

<ol class="bu1-template-list">
  <li>tvrdý první řádek</li>
  <li>1 až 3 krátké věty s pointou</li>
  <li>konkrétní CTA</li>
</ol>

## Pravidla

<ul class="bu1-template-list">
  <li>první řádek musí fungovat samostatně</li>
  <li>používej krátké odstavce</li>
  <li>nepiš dlouhé storytelling intro</li>
  <li>emoji drž účelně, obvykle 1 až 3</li>
  <li>hashtagy drž střídmě</li>
  <li>produkt nebo akce musí být jasné do pár sekund</li>
</ul>

## Směry

<ul class="bu1-template-list">
  <li>novinka nebo drop</li>
  <li>připomenutí benefitu</li>
  <li>jasný drop nebo termín</li>
  <li>krátká reference nebo důkaz z praxe</li>
</ul>

## Čemu se vyhnout

<ul class="bu1-template-list">
  <li>generický motivační text</li>
  <li>emoji bez funkce</li>
  <li>nejasná pointa bez CTA</li>
  <li>stylizovaný text bez vazby na produkt nebo informaci</li>
</ul>
